# Changelog

All notable changes to this project are documented in this file.

This project follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)

## [0.4.0] - 2026-04-17

### Added
- Organize mode improvements for copy/move workflows.
- Name conflict dialog with rename/overwrite/cancel.
- UX upgrades: mode banner, inline hints, toasts, operation log overlay.
- Browser filter, onboarding dialog, customizable hotkeys.
- Sorting rules dry-run + execution and review labels export.
- Public-repo readiness files (license, CI, contribution templates).
